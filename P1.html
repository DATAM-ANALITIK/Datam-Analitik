Descripción

1 DEEP LEARNING

Deep Learning o tambien conocido como aprendizaje automático, es una de las ramas de la Ciencia de Datos e Informatica, que esta conformada por algoritmos de aprendizaje no supervisado, es decir estos aprenden solos, muchas son las aplicaciones de este tipo de ramas, como lo es el reconocimiento facial, patrones de compra, predicciones de clima o consumo entre otras.

2 APLICACIONES

En el Sector Comercial: Empresas como Amazon, Netflix, utilizan esta metodología para generar sistemas de recomendaciones de esta forma cada vez que ingresamos a estas plataformas nos sugieres películas y series.

En el Sector Financiero: Utilizan esta técnologia para detectar fraudes de tarjeta de creditos, tambien patrones de comportamiento de cliente para realizar sugerencias de consumos.

En Marketing: Determina patrones de consumo por distintas variables de estudio, como colores, genero, ingreso entre otras, dando el perfil del cliente potencial.
