Descripción

1 MACHINE LEARNING

Machine Learning o aprendisaje automatico supervisado, es otra de las grande ramas de la Ciencia de Datos a diferencia del Deep Learning este requiere la supervisión de un analista para verificar su funcionamiento, las aplicaciones son varias dentro de todas las ramas de la cienca incluidas las cualitativas.

2APLICACIONES

☼ MEDICINA: Actualmente este tipo de aplicación ayuda a la desición medica, mediante procesamiento de bases de datos de historias clinicas registradas puede detectar enfermedades. En la actualidad se utiliza para la detección de cancer y otras enfermedades prexistentes.

☼ BIG DATA: Con la generacion continua de información de todo tipo es casi imposible clasificar o encontrar lo que requerimos para nuestra insdustria, en este aspecto Machine Learning ingresa como una herramienta clave para poder encontrar estos patrones de comportamiento aplicando algorítmos de Minería de Datos para encontrar estos patrones y determinar soluciones claves de Inteligencia de Negocios, entre otras.

☼ NEGOCIOS: Nos ayuda a detectar clientes potenciales y su ubicación, no solo por medio de encuestas si no por redes sociales como Instagram, Facebook o Twitter, determinando cualos son los gustos de los clientes, esta tecnologia la aplica las redes mencionadas, para determinar sugerencias, por tal motivo cada vez que buscamos algun producto en Facebook, este se encarga de mostrate más información sobre ese tema.

☼ FINANZAS: Nos ayuda a determinar nuestros clientes potenciales, capacidad de colocación de crédito, tambien la permanencia del cleinte o a su vez cual es el perfil de los clientes que mas creditos realizan, tambien aplicaciones mas complejas como detención de fraudes entre otras.
